# Task Scheduling System

## Overview
Task Scheduling System is a Java console application developed for the IT245 Data Structures course at the Saudi Electronic University. The project demonstrates how different data structures can be used to manage and schedule tasks efficiently.

## Features

- Add new tasks
- Remove existing tasks
- Search for tasks
- Update task information
- Display all scheduled tasks
- Execute task scheduling based on priority
- Compare different data structure implementations

## Data Structures Used

This project implements two different approaches:

- ArrayList
- Priority Queue (or HashMap if your project uses HashMap)

The purpose is to compare their performance and time complexity.

## How to Run

1. Clone the repository

```
git clone https://github.com/Samer-Alsaykhan/Task-Scheduling-System.git
```

2. Open the project in Visual Studio Code.

3. Compile the project

```
javac *.java
```

4. Run the application

```
java Main
```
